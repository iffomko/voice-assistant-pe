\doxysubsection{Detailed Description}
Этот класс отвечает за очистку сообщений в объектах \doxylink{classcom_1_1iffomko_1_1voice_assistant_1_1_a_p_is_1_1open_a_i_1_1_a_i_service}{AIService}. Он очищает сообщения если срок последнего взаимодействия в этом чате на текущий момент больше 1 часа. 

Этот класс реализуется интерфейс {\ttfamily Runnable}, следовательно его нужно запускать в отдельном потоке, потому что иначе программа просто зависнет.
